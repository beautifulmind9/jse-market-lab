# UX Learnings

## Key usability issues observed

### 1. Product identity confusion
Some users interpret the dashboard as a static report instead of a tool they interact with.

Implication:
The product must clearly show where user input affects the output and reinforce that this is a planning tool.

---

### 2. Capital input confusion
Users may not immediately understand that the capital field represents their own money.

Implication:
Labeling and helper text must clearly explain the purpose of the input.

---

### 3. System behavior clarity
Users ask whether the system updates or adapts after trades are followed.

Implication:
The platform should clearly explain that outputs are based on current available data and can be refreshed as new data becomes available.

---

### 4. Tab flow expectations
Users expect a logical flow that matches how they think:
- understand the tool
- see opportunities
- plan
- review

Implication:
Tab order and navigation should reflect this mental model.

---

## What is working well

### Guided view
- easy to understand
- not overwhelming
- supports beginners effectively

### Advanced view
- correctly perceived as deeper analysis
- appropriate for more experienced users

### Ticker analysis
- intuitive
- engaging
- strong entry point into the product

---

## Core UX goal
Shift user mindset from:

"This is interesting"

To:

"This helps me decide what to do"

---

## UX design principles going forward
- show the most important information first
- keep explanations clear and specific
- avoid long blocks of visible text
- move secondary content into expanders or optional sections
- reinforce what the user can do with the information
- ensure every section answers a clear question

---

## Validation approach
Ongoing user testing should focus on:
- where users hesitate
- what they misunderstand
- what they ignore
- what they naturally click first
- what increases trust

---

## Key insight
The product is structurally strong.

The main opportunity now is improving clarity and guiding user behavior more effectively.
