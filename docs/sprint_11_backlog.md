# Sprint 11 — Backlog

## EPIC: Review & Discipline Layer

---

### Story 11.1 — Trade Review Engine
**As a user**
I want to see if my trades followed system rules
So I can understand if I made disciplined decisions

**Acceptance Criteria**
- each trade has a review record
- includes rule adherence flags
- includes rank comparison

---

### Story 11.2 — Mistake Detection
**As a user**
I want to see where I made mistakes
So I can avoid repeating them

**Acceptance Criteria**
- detects ignored higher-rank trades
- detects low-quality trades
- detects liquidity violations
- no duplicate mistakes

---

### Story 11.3 — Behavior Summary
**As a user**
I want a simple summary of my behavior
So I can quickly understand patterns

**Acceptance Criteria**
- 2–4 insights generated
- plain language
- no advisory tone

---

### Story 11.4 — Discipline Score
**As a user**
I want a score reflecting my discipline
So I can track consistency

**Acceptance Criteria**
- score between 0–100
- based on rule adherence
- stable across runs

---

### Story 11.5 — UI Integration
**As a user**
I want to see my decision review clearly
So I can interpret it easily

**Acceptance Criteria**
- Decision Review section exists
- includes summary, mistakes, score
- no clutter or duplication
