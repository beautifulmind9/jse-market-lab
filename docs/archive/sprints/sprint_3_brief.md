# Sprint 3 Brief

## Sprint Name
Jamaican Clarity Layer

## Goal
Make decision guidance easier to understand for the average Jamaican user.

## In Scope
- Update guidance wording
- Add Clear and Pro message bodies
- Add UI selection for wording mode
- Preserve current logic and behavior

## Out of Scope
- New ranking logic
- New warning logic
- New earnings logic
- Monetization work
- Styling overhaul

## Definition of Done
- Guidance supports Clear and Pro wording
- Clear wording feels natural and easy to understand
- Pro wording stays concise and locally readable
- UI defaults to Clear
- Tests pass
