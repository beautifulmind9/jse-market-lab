# Sprint 6 Requirements — Analyst Intelligence Layer

## Functional Requirements
1. The dashboard must provide an Analyst Insights section.
2. The section must include Feature Insights, Performance Matrix, and Exit Analysis.
3. Feature summaries must include count, win rate, average return, and median return.
4. Performance Matrix must summarize results by quality tier and holding window when available.
5. Exit Analysis must summarize results by exit_reason when available.
6. Missing columns must not crash the dashboard.

## Non-Functional Requirements
- Analyst-only placement
- Readable table-first UI
- No modifications to core decision engines

## Definition of Done
- Analyst Insights section implemented
- Missing-field handling works
- Tests pass
