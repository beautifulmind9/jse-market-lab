# Sprint 12 Definition of Done

Sprint 12 is done when the following conditions are true:

## Public-facing baseline
- the app can be deployed and accessed publicly
- the app no longer feels like an internal-only prototype

## Data trust
- bundled JSE dataset loads by default
- active dataset source is visible
- canonical ticker behavior is stable
- marker variants do not split the ticker universe

## Stability
- app startup is resilient
- missing legacy files do not crash the app
- invalid zero-entry-price paths do not break return calculations

## User experience
- first-run section explains the product clearly
- beginner mode is lighter than analyst mode
- technical/raw content is isolated appropriately
- Ticker Analysis explains meaning before detail

## Phase 1 readiness
- the app is stable enough to share as a public Phase 1 product surface
- remaining gaps are explanation-layer and UX-polish gaps, not foundational system failures
