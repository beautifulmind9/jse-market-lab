# Sprint 11 — User Flow

## Entry Point
User navigates to:
→ "Decision Review" tab

## Flow

1. System loads recent trades
2. Trade Review layer evaluates each trade:
   - compares against signals
   - checks rule adherence

3. Mistake Detection runs:
   - identifies deviations
   - aggregates issues

4. Behavior Summary generates:
   - 2–4 insights based on patterns

5. Discipline Score is computed

## Output Sections

### 1. Behavior Summary
- short bullet insights

### 2. Mistakes
- list of detected issues

### 3. Discipline Score
- numeric indicator

### 4. Trade Breakdown (optional)
- per-trade evaluation

## Exit
User leaves with:
- awareness of mistakes
- understanding of behavior patterns
