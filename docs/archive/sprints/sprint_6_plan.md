# Sprint 6 — Analyst Intelligence Layer

## Goal
Add analyst-facing insights that help validate and explain system behavior without changing the core decision engine.

## Features
- Feature Insights
- Performance Matrix
- Exit Analysis

## Outcome
Moves the product from decision-support only to decision-support plus analyst validation.

## In Scope
- grouped feature summaries
- tier × holding-window performance summaries
- exit-reason analysis
- analyst-only UI tab

## Out of Scope
- signal changes
- confidence changes
- allocation changes
- AI explanations
