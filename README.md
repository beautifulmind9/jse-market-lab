# JSE Market Lab

An educational market-insight and analytics framework for understanding the structure and behaviour of the Jamaica Stock Exchange (JSE).

## What this project is
- A **decision-support education tool** for Caribbean investors and analysts
- A **market-structure and trading-behaviour lab** (breadth, liquidity conditions, signal distributions, risk framing)
- A place to test **frameworks** that help investors think clearly (not chase trades)

## What this project is not
- Not an investment advisory service
- Not a signal-selling product
- **Does not scrape or redistribute JSE exchange data**
- Does not provide “buy/sell” recommendations

## Data policy (important)
This repository is designed to operate using:
- **Synthetic / demo datasets**, and/or
- **User-provided datasets** that the user lawfully obtained, and/or
- **Licensed datasets** (where applicable)

No exchange website scraping logic or exchange-distributed raw datasets are included in this public repository.
